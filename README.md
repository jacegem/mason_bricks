# Mason Bricks

Flutter 프로젝트에서 사용하는 bricks 입니다.

## brick 생성

```shell
mason new page -o bricks 
```

## brick 추가

```shell
mason add page --path bricks/page 
```

## brick 으로 만들기

```shell
mason make page -o examples
```
